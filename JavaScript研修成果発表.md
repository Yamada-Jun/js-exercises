# JavaScript研修成果発表

## 発表内容

最終回では成果発表として、**振り返りのプレゼンテーション**と**最終課題のデモ**を行ってもらいます。

各人の持ち時間は両方合計で**20分**です。

### 振り返りのプレゼンテーション

次の事柄が含まれるようにしてください(それぞれ、最低1ページ)。

- 研修の成果(研修を通して学んだこと)
- 研修で苦労したこと
- 今後研修する人達へのアドバイス
- 講師陣へのアドバイス
- 研修成果の今後への活用

### 最終課題のデモ

会心のデジタル時計の作成をお願いします。

最終課題は、以下の点を満たしていれば何でもよいです。

- JavaScriptまたはTypeScriptで書かれていること
- 時計が何らかの形で、最終的には表示されること

時計のデモで何らかの特別な機材が必要な場合には、各人で準備してください。
